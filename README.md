# 🚀 FinTrack — Landing Page

A modern, minimal, and premium landing page for **FinTrack**, a smart personal finance app.
Built with **Next.js**, **Tailwind CSS**, and **shadcn/ui**, focusing on clean UI, smooth UX, and App Store–level presentation.

---

## ✨ Overview

FinTrack helps users:

- Track expenses effortlessly
- Manage budgets intelligently
- Visualize financial growth

This landing page showcases the app with a **polished, product-first design** inspired by Apple & Google Play aesthetics.

---

## 🧩 Features

- 🧭 **Responsive Navbar** (clean, floating, glass effect)
- 🎯 **Hero Section** with strong typography & CTA
- 📱 **App Screenshots Grid** (App Store style layout)
- ⚡ **Interactive Modal** (App status + contact)
- 🧱 **Feature Sections** (modular & reusable)
- 🧾 **Developer Credit Strip** (premium branding touch)
- 🔗 **Footer with Social Links**
- 🌗 Subtle gradients, grid backgrounds & soft shadows

---

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Icons:** Lucide & Phosphor Icons
- **Fonts:** Geist (via next/font)

---

## 📁 Project Structure

```
app/            → App router (layout, page, global styles)
components/     → All UI + sections (modular components)
components/ui/  → Reusable UI primitives (shadcn)
public/         → Static assets (images, icons)
lib/            → Utilities
```
